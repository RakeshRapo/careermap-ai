# CareerMap AI

AI-powered career guidance platform built using Spring Boot and PostgreSQL.

## Features

* User Registration
* User Profile Management
* Career Roadmap Generation
* Multi-Agent Architecture
* Gemini AI Integration (In Progress)

## Tech Stack

* Java 21
* Spring Boot
* PostgreSQL
* Maven
* Lombok
* Git & GitHub

## Future Features

* Resume Analysis Agent
* DSA Mentor Agent
* Interview Coach Agent
* AI-Powered Personalized Roadmaps
